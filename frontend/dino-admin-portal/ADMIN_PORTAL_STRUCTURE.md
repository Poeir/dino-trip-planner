# Dino Admin Portal - โครงสร้างหน้าเว็บ

## 📋 รายการหน้าเว็บที่สร้างแล้ว

### 1. 📊 Dashboard (แดชบอร์ด)
**Path:** `/dashboard`  
**File:** `src/page/DashboardPage.jsx`

**ฟีเจอร์:**
- แสดงสถิติข้อมูลทั้งหมด (สถานที่, กิจกรรม, อีเวนต์, โปรแกรมท่องเที่ยว)
- Quick Actions สำหรับเพิ่มข้อมูลแต่ละประเภท
- กิจกรรมล่าสุดในระบบ
- สถานะการทำงานของระบบ

---

### 2. 📍 Places (จัดการสถานที่)
**Path:** `/places`  
**File:** `src/page/PlacesPage.jsx`

**ฟีเจอร์:**
- แสดงรายการสถานที่ทั้งหมดในรูปแบบตาราง
- ค้นหาสถานที่ตามชื่อ
- กรองตามประเภท (tourist_attraction, restaurant, hotel, museum, park)
- แสดงข้อมูล: ชื่อ, ประเภท, คะแนน, สถานะ
- ปุ่มจัดการ: แก้ไข, ลบ

**Schema ที่ใช้:** `backend/models/Place.js`
- Google Place ID
- Core info (name, type, location, rating)
- Contact & Address
- Opening Hours
- Media (photos)
- Reviews
- Features (takeout, delivery, etc.)

---

### 3. 🎯 Activities (จัดการกิจกรรม)
**Path:** `/activities`  
**File:** `src/page/ActivitiesPage.jsx`

**ฟีเจอร์:**
- แสดงรายการกิจกรรมในรูปแบบ Grid Cards
- ค้นหากิจกรรมตามชื่อ
- กรองตามหมวดหมู่ (outdoor, food, culture, shopping, wellness, entertainment, sports, education)
- แสดงรูปภาพปก, หมวดหมู่, คำอธิบาย, Tags, Suitable For
- นับจำนวนสถานที่ที่เกี่ยวข้อง
- ปุ่มจัดการ: แก้ไข, ลบ

**Schema ที่ใช้:** `backend/models/Activity.js`
- Name & Slug
- Category
- Description & Cover Image
- Places (สถานที่ที่ทำกิจกรรมได้)
- Tags
- Suitable For (solo, couple, family, group, elderly)

---

### 4. 🎉 Events (จัดการอีเวนต์)
**Path:** `/events`  
**File:** `src/page/EventsPage.jsx`

**ฟีเจอร์:**
- แสดงรายการอีเวนต์ในรูปแบบ List View
- ค้นหาอีเวนต์ตามชื่อ
- กรองตามหมวดหมู่ (concert, festival, exhibition, sport, market, workshop, religious, food)
- กรองตามสถานะ (upcoming, ongoing, past)
- แสดงรูปภาพปก, วันที่, สถานที่จัด, ค่าเข้างาน
- Status Badge แสดงสถานะของอีเวนต์
- ปุ่มจัดการ: ดูรายละเอียด, แก้ไข, ลบ

**Schema ที่ใช้:** `backend/models/Event.js`
- Name & Slug
- Category
- Description & Images
- Venues (สถานที่จัดงาน)
- Schedule (วันเวลา, sessions)
- Admission (ค่าเข้างาน, tickets)

---

### 5. 🗺️ Itineraries (จัดการโปรแกรมท่องเที่ยว)
**Path:** `/itineraries`  
**File:** `src/page/ItinerariesPage.jsx`

**ฟีเจอร์:**
- แสดงรายการโปรแกรมท่องเที่ยว
- ค้นหาตามชื่อหรือธีม
- กรองตามผู้สร้าง (AI/LLM หรือ User)
- กรองตามสถานะ (สาธารณะ/ส่วนตัว)
- แสดง Timeline จุดหมาย (แสดงสูงสุด 5 จุดแรก)
- แสดงสถิติ: จำนวนจุดแวะ, กิจกรรม, วันที่สร้าง
- ปุ่มจัดการ: ดูรายละเอียด, แก้ไข, เผยแพร่/ซ่อน, ลบ

**Schema ที่ใช้:** `backend/models/Itinerary.js`
- Title & Theme
- Duration (days)
- Stops (จุดหมายพร้อม order, arrival time, duration)
- Generated By (llm/user)
- Is Public
- User ID
